# Early Church Documents PDFs

This directory should contain the PDF files for the Early Church Timeline documents.

## Required PDFs:

1. `Letter_of_Barnabas.pdf` - Letter of Barnabas (c. 75 AD)
2. `Clement_of_Rome.pdf` - First Epistle of Clement to the Corinthians (c. 80 AD)
3. `Ignatius_of_Antioch.pdf` - Letters of Ignatius of Antioch (c. 107 AD)
4. `Polycarp_Epistle_Philippians.pdf` - Epistle to the Philippians (c. 110–140 AD)
5. `Hegesippus_Memoirs.pdf` - Memoirs of Hegesippus (c. 130 AD)
6. `Martyrdom_of_Polycarp.pdf` - Martyrdom of Polycarp (c. 155 AD)
7. `Justin_martyr.pdf` - First Apology & Dialogue with Trypho (c. 155–165 AD)
8. `Tatian_Diatessaron.pdf` - Diatessaron (c. 160–175 AD)
9. `To_Diognetus.pdf` - To Diognetus (c. 160 AD)
10. `Dionysius_of_Corinth.pdf` - Letter to Pope Soter (c. 170 AD)
11. `Irenaeu.pdf` - Against Heresies (c. 180 AD)
12. `Clement_of_Alexandria_Stromata.pdf` - The Stromata (c. 190–200 AD)

## Sources:

These documents can be obtained from:

- Early Christian Writings (www.earlychristianwritings.com)
- New Advent Church Fathers collection (www.newadvent.org/fathers)
- Christian Classics Ethereal Library (www.ccel.org)

Place all PDF files in this `/public/pdfs/` directory with the exact filenames listed above.
